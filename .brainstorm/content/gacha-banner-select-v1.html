④ СПЕКА (черновик)

Общее для всех вариантов

Баннер получает backdrop_path (отдельная загрузка в админке, kind=banners). Слайды/карточки используют backdrop как фон (cover) + нижний скрим rgba(8,8,15,.15→.92); стандартный баннер всегда первый; ивенты — по sort_order; бейдж СТАНДАРТНЫЙ/ИВЕНТ·до даты. Экран крутки /gacha/:id тоже получает этот backdrop фоном (блюр 24px + скрим) вместо чёрного.

A — hero-слайдер

Высота ~340px desktop / 260px mobile, стрелки + точки + свайп + автоплей 7с (pause on hover). CTA «Крутить» → /gacha/:id.

B — лента

Карточки 340×200, snap-scroll. Дёшево, читаемо, видны тизеры «СКОРО».

C — всё-в-одном

Слайдер A + закреплённая панель крутки (pity + ×1/×10) и секции пула/результатов под ней. /gacha/:id удаляется, роут /gacha/:id редиректит на /gacha?banner=:id.
