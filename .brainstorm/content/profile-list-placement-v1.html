Список аниме ↔ витрина — 3 варианта расположения

Проблема: большая мозаика-витрина оттолкнула основной вотчлист («Список») далеко вниз. Ниже — 3 способа их подружить. Контент условный, токены реальные.

1Витрина = первый таб рекомендую

Под шапкой сразу таб-бар: Витрина · Список · Отзывы… Витрина открыта по умолчанию, но «Список» — соседний таб: клик → вотчлист сразу под табами, без долгого скролла. Паттерн MAL / AniList / Steam.

↳ кликни табы «Витрина»/«Список» в макете — переключаются по-настоящему

  • список всегда в один тап, без скролла
  • масштабируется на любое число табов
  • витрина = «лицо», список = функция, чётко разделены
  • гость видит витрину, список — по клику

2Компактная витрина + раскрытие

Витрина остаётся над табами, но по умолчанию обрезана до ~2 рядов с затуханием и кнопкой «Показать всю витрину». Табы и «Список» поднимаются заметно выше — баланс «видно витрину сразу, но она не доминирует».

3Sticky-табы (минимум правок)

Порядок не меняем — витрина сверху во всю красу. Но таб-бар «прилипает» к верху при скролле, так что «Список» всегда в один тап, где бы ты ни был. Самая дешёвая правка.
