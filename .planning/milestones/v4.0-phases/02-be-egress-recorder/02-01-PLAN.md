## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| BE service → analytics `/internal/effects` | Internal Docker-network producer→sink; untrusted only if the network is breached |
| recorder ctx → outbound 3rd-party wire | OTel baggage is injected on outbound requests; PII here leaks to external hosts |
| browser/gateway → BE inbound | seeds baggage; the only place user_id enters context |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-PII | Information Disclosure | recording RoundTripper / baggage | mitigate | user_id carried via a PRIVATE non-propagated ctx value (WithUserID), NOT W3C baggage; only origin/operation in wire baggage. Full strip + TestNoUserIDOnOutboundWire lands in Plan 02-04. |
| T-02-INT | Elevation/Tampering | POST /internal/effects | mitigate | Registered only in analytics router beside /internal/erase; never gateway-proxied (Docker-network-only). |
| T-02-VAL | Tampering | /internal/effects body | mitigate | 256KB io.LimitReader (mirror collect.go) + bound array length; malformed JSON → 400. |
| T-02-INJ | Tampering | ClickHouse effect insert | mitigate | Parameterized PrepareBatch.Append in exact column order; no string interpolation (unchanged from Phase 1). |
| T-02-DOS | Denial of Service | producer ring buffer | mitigate | Non-blocking drop-on-full Record + dropped counter; producer never blocks the outbound hot path (D-10). |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | No new external packages this phase (RESEARCH §Package Legitimacy Audit: zero registry installs; baggage is a sub-pkg of existing otel). |
