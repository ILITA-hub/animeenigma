    4. **Smoke tests against the live deployment:**
       ```bash
       # 4a — counter families exist in /metrics
       curl -s http://localhost:8083/metrics | grep -E "^# HELP combo_(override|resolve)_total" | wc -l
       # Expected: 2

       # 4b — anon resolve smoke
       curl -sw "\n%{http_code}\n" -X POST http://localhost:8000/api/preferences/resolve \
         -H "Content-Type: application/json" \
         -H "X-Anon-ID: smoke-test-$(date +%s)" \
         -d '{"anime_id":"smoke-test-anime","available":[{"player":"kodik","language":"ru","watch_type":"sub","translation_title":"AniLibria"}]}' \
         | tail -1
       # Expected: 200

       # 4c — anon override smoke
       SMOKE_ID=$(uuidgen 2>/dev/null || echo "smoke-$(date +%s)-aaaa-bbbb-cccc-dddddddddddd")
       BEFORE=$(curl -s http://localhost:8083/metrics | awk '/^combo_override_total\{/ { sum += $NF } END { print sum+0 }')
       curl -sw "\n%{http_code}\n" -X POST http://localhost:8000/api/preferences/override \
         -H "Content-Type: application/json" \
         -H "X-Anon-ID: $SMOKE_ID" \
         -d '{"anime_id":"smoke-test","load_session_id":"'"$SMOKE_ID"'","dimension":"language","original_combo":{"language":"ru","player":"kodik"},"new_combo":{"language":"en"},"ms_since_load":2500,"tier":"default","tier_number":5,"player":"kodik"}' \
         | tail -1
       # Expected: 204

       AFTER=$(curl -s http://localhost:8083/metrics | awk '/^combo_override_total\{/ { sum += $NF } END { print sum+0 }')
       echo "before=$BEFORE after=$AFTER"
       # Expected: after > before (counter incremented)

       # 4d — invalid dimension smoke
       curl -sw "\n%{http_code}\n" -X POST http://localhost:8000/api/preferences/override \
         -H "Content-Type: application/json" \
         -H "X-Anon-ID: $SMOKE_ID-2" \
         -d '{"anime_id":"x","load_session_id":"y","dimension":"quality","original_combo":null,"new_combo":{},"ms_since_load":1000,"tier":"default","tier_number":5,"player":"kodik"}' \
         | tail -1
       # Expected: 400 (rejects invalid dimension per T-01-02)

       # 4e — both-missing smoke
       curl -sw "\n%{http_code}\n" -X POST http://localhost:8000/api/preferences/override \
         -H "Content-Type: application/json" \
         -d '{"anime_id":"x","load_session_id":"y","dimension":"language","original_combo":null,"new_combo":{},"ms_since_load":1000,"tier":"default","tier_number":5,"player":"kodik"}' \
         | tail -1
       # Expected: 400 (rejects request without JWT and without X-Anon-ID per T-01-01)
       ```

    5. **Grafana dashboard reload** (Pitfall 4):
       After confirming smoke tests pass, restart Grafana to pick up the new dashboard JSON immediately (the file provider polls every ~10s but explicit restart guarantees pickup):
       ```bash
       docker compose -f docker/docker-compose.yml restart grafana
       sleep 10
       docker compose -f docker/docker-compose.yml logs grafana --tail 20 | grep -i "provision\|reload"
       ```

    Save the smoke test outputs to `/tmp/plan07-smoke.log` for the SUMMARY.

    If ANY smoke test fails, STOP. Do not proceed to the human-verify checkpoint or the changelog. Diagnose via `make logs-<service>`, fix, redeploy, retry.
