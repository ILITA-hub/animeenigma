<objective>
Land the data-layer foundation that Phase 12 needs to ship S5: extend the animes schema with the six new attribute dimensions (kind / rating / material_source / studios / tags), expand the Shikimori parser GraphQL queries to populate kind / rating / material_source / studios on every fetch, and add a new AniList GraphQL client that exposes a single FetchTags method ready for the Wave-2 backfill to drive. Anime persisted by this plan still feed Phase 11's "Up Next for you" row unchanged — only the schema is enriched. Producers are absorbed into studios per CONTEXT.md Decision §A2 — there is no separate producers column or m2m table in this plan and no future plan adds one until v3.0.

Purpose: Phase 11's SUMMARY explicitly handed the schema gap to Phase 12. Without these schema additions, S5 has no data to TF-IDF over for five of its six attribute dimensions (only `genres` exists today). This plan is a Wave-1 prerequisite for the Wave-2 backfill (which populates existing anime rows) and the Wave-3 S5 implementation (which reads from the populated schema). Splitting the work this way keeps each commit boundary clean: schema lands in the catalog service alone, backfill is a maintenance-service job, S5 is a player-service signal — three services, three plans, three deploy boundaries.

Output:
- services/catalog/internal/domain/anime.go — Anime struct with Kind, Rating, MaterialSource string columns + Studios []Studio and Tags []Tag m2m relations. New Studio, Tag, and AnimeTag (join with Rank) struct types.
- services/catalog/internal/parser/shikimori/client.go — GraphQL queries + struct definitions + mapping functions extended for kind, rating, source, studios { id name } across all 6 fetch paths.
- services/catalog/internal/parser/anilist/client.go (NEW) — Thin GraphQL client with FetchTags(ctx, anilistID int). Rate-limited at ~1 rps unauthenticated.
- services/catalog/internal/parser/anilist/client_test.go (NEW) — Table-driven tests via httptest covering happy path, empty tags, GraphQL errors, network failure, malformed JSON, and slugifyTagName edge cases.
- services/catalog/cmd/catalog-api/main.go — AutoMigrate extended to include &domain.Studio{} and &domain.Tag{}; SetupJoinTable registers the AnimeTag explicit join. The join tables anime_studios and anime_tags are auto-created by GORM from the m2m struct tags.

After this plan ships, make redeploy-catalog brings up a service with the enriched schema (\\d animes shows the new columns; \\dt anime_studios anime_tags studios tags shows the new tables) and freshly-fetched anime carry the new fields. Existing rows are NOT yet populated — the Wave-2 backfill plan runs against this schema.
</objective>

Six new dimensions on `animes`:
- kind (string, size:20, indexed) — TV / Movie / OVA / ONA / Special / Music
- rating (string, size:20) — G / PG / PG-13 / R / R+ / Rx (the S5 "demographic" proxy per Decision §A3)
- material_source (string, size:50) — original / manga / novel / light_novel / visual_novel / game / web_manga / etc. Column named material_source to avoid collision with the existing SourceType constants on Video.
- studios m2m — Studio { id PK string, name string, created_at } + anime_studios join. Studio.ID is the Shikimori studio id.
- tags m2m — Tag { id PK string, name string, source string default 'anilist', created_at } + AnimeTag { anime_id, tag_id, rank } join. Tag.ID is the slugified AniList tag name (deterministic + idempotent).
- Producers are intentionally absent (Decision §A2 collapses producers into studios — single dimension, weight 0.25 in S5).

Shikimori GraphQL fields used: kind, rating, source, studios { id name }. Verified in Shikimori's GraphQL playground per CONTEXT.md §specifics.

AniList endpoint: https://graphql.anilist.co (POST). No auth required. Rate limit: 90 req/min unauthenticated. AniList query body:
  query ($id: Int) {
    Media(id: $id, type: ANIME) {
      tags { name rank category isAdult isGeneralSpoiler }
    }
  }

AniList rank scale: 0-100. Decision §A4 says S5 v1 ignores rank but we persist it on AnimeTag.Rank for future use.

AniList isAdult / isGeneralSpoiler fields: persisted in the parser-local Tag struct but the Wave-2 backfill chooses whether to filter by them. This plan only fetches.

The new tables (studios, tags, anime_studios, anime_tags) are auto-created by GORM AutoMigrate on the next catalog restart. CLAUDE.md confirms GORM ADDS but never DROPS — safe forward migration.

Open verification items (decided in plan-phase, recorded for execute-phase):
1. Tag.ID format: AniList does NOT expose a stable integer id for tags in the public schema — only name is stable. We use the lower-cased name with non-alphanumerics replaced by underscore as the primary key (deterministic, idempotent). Concrete: slugify("Slice of Life") returns "slice_of_life". Function lives in the AniList parser package as slugifyTagName(name string) string.
2. Studio.ID format: Shikimori DOES expose stable integer studio ids. Use the string form directly, mirroring Genre.ID.
3. AniList user-agent: Set to "AnimeEnigma/1.0 (https://animeenigma.ru)" per AniList ToS recommendation. Mirror the Shikimori userAgentTransport pattern.
4. AniList rate limit headroom: 90 req/min observed limit. We configure the local rateLimiter with rps = 1 (1 req/sec = 60 req/min, leaves 33% headroom for retries). Backfill in Wave 2 can raise this if it parallelizes via a separate token bucket.
5. GORM JoinReferences pattern: declare AnimeTag struct with composite PK + Rank; on the Anime side use `gorm:"many2many:anime_tags;joinForeignKey:AnimeID;joinReferences:TagID"`; in main.go after AutoMigrate call db.SetupJoinTable(&Anime{}, "Tags", &AnimeTag{}) to register the explicit join model.

       ```go
       package anilist

       import (
           "bytes"
           "context"
           "encoding/json"
           "fmt"
           "net/http"
           "regexp"
           "strings"
           "sync"
           "time"

           "github.com/ILITA-hub/animeenigma/libs/errors"
           "github.com/ILITA-hub/animeenigma/libs/logger"
       )

       const (
           defaultEndpoint  = "https://graphql.anilist.co"
           defaultUserAgent = "AnimeEnigma/1.0 (https://animeenigma.ru)"
           // 1 rps = 60 req/min — leaves 33% headroom under AniList's 90/min cap.
           defaultRateLimitRPS = 1
           defaultTimeout      = 10 * time.Second
       )

       // Tag is the parser-local representation of an AniList tag.
       // Rank is preserved for v2.1 rank-weighted TF-IDF (Decision §A4 ignores
       // rank in v1). Category / IsAdult / IsGeneralSpoiler are exposed for
       // the Wave-2 backfill to filter on if it chooses; this client does
       // not filter on them.
       type Tag struct {
           Name             string
           Rank             int
           Category         string
           IsAdult          bool
           IsGeneralSpoiler bool
       }

       // Client for AniList GraphQL public API.
       type Client struct {
           httpClient  *http.Client
           endpoint    string
           userAgent   string
           log         *logger.Logger
           rateLimiter *rateLimiter
       }

       type rateLimiter struct {
           mu         sync.Mutex
           tokens     int
           maxTokens  int
           lastRefill time.Time
           interval   time.Duration
       }

       func newRateLimiter(rps int) *rateLimiter {
           return &rateLimiter{tokens: rps, maxTokens: rps, lastRefill: time.Now(), interval: time.Second}
       }

       func (rl *rateLimiter) acquire() {
           rl.mu.Lock()
           defer rl.mu.Unlock()
           now := time.Now()
           elapsed := now.Sub(rl.lastRefill)
           if elapsed >= rl.interval {
               rl.tokens = rl.maxTokens
               rl.lastRefill = now
           }
           if rl.tokens <= 0 {
               time.Sleep(rl.interval - elapsed)
               rl.tokens = rl.maxTokens
               rl.lastRefill = time.Now()
           }
           rl.tokens--
       }

       // NewClient constructs a production AniList client.
       func NewClient(log *logger.Logger) *Client {
           return NewClientWithBaseURL(defaultEndpoint, log)
       }

       // NewClientWithBaseURL allows tests to point the client at a stub server.
       func NewClientWithBaseURL(endpoint string, log *logger.Logger) *Client {
           return &Client{
               httpClient:  &http.Client{Timeout: defaultTimeout},
               endpoint:    endpoint,
               userAgent:   defaultUserAgent,
               log:         log,
               rateLimiter: newRateLimiter(defaultRateLimitRPS),
           }
       }

       const fetchTagsQuery = `query ($id: Int) {
         Media(id: $id, type: ANIME) {
           tags { name rank category isAdult isGeneralSpoiler }
         }
       }`

       // FetchTags returns AniList tags for the given anime by its AniList id.
       // Decision §A4: rank is preserved on the returned Tag.Rank but the v1 S5
       // signal ignores it. isAdult / isGeneralSpoiler are returned for caller
       // filtering — this method does not filter.
       func (c *Client) FetchTags(ctx context.Context, anilistID int) ([]Tag, error) {
           c.rateLimiter.acquire()

           reqBody := map[string]interface{}{
               "query":     fetchTagsQuery,
               "variables": map[string]interface{}{"id": anilistID},
           }
           jsonBody, err := json.Marshal(reqBody)
           if err != nil {
               return nil, errors.ExternalAPI("anilist", err)
           }

           req, err := http.NewRequestWithContext(ctx, "POST", c.endpoint, bytes.NewBuffer(jsonBody))
           if err != nil {
               return nil, errors.ExternalAPI("anilist", err)
           }
           req.Header.Set("Content-Type", "application/json")
           req.Header.Set("Accept", "application/json")
           req.Header.Set("User-Agent", c.userAgent)

           resp, err := c.httpClient.Do(req)
           if err != nil {
               return nil, errors.ExternalAPI("anilist", err)
           }
           defer resp.Body.Close()

           var result struct {
               Data struct {
                   Media *struct {
                       Tags []struct {
                           Name             string `json:"name"`
                           Rank             int    `json:"rank"`
                           Category         string `json:"category"`
                           IsAdult          bool   `json:"isAdult"`
                           IsGeneralSpoiler bool   `json:"isGeneralSpoiler"`
                       } `json:"tags"`
                   } `json:"Media"`
               } `json:"data"`
               Errors []struct {
                   Message string `json:"message"`
               } `json:"errors"`
           }

           if err := json.NewDecoder(resp.Body).Decode(&result); err != nil {
               return nil, errors.ExternalAPI("anilist", err)
           }

           if len(result.Errors) > 0 {
               return nil, errors.ExternalAPI("anilist", fmt.Errorf("%s", result.Errors[0].Message))
           }

           if result.Data.Media == nil {
               return []Tag{}, nil
           }

           out := make([]Tag, 0, len(result.Data.Media.Tags))
           for _, t := range result.Data.Media.Tags {
               out = append(out, Tag{
                   Name:             t.Name,
                   Rank:             t.Rank,
                   Category:         t.Category,
                   IsAdult:          t.IsAdult,
                   IsGeneralSpoiler: t.IsGeneralSpoiler,
               })
           }
           return out, nil
       }

       // slugifyTagName converts an AniList tag name into a deterministic,
       // lowercase, alphanumeric+underscore primary key. Used by the Wave-2
       // backfill to populate domain.Tag.ID. Multiple consecutive non-alnum
       // characters collapse to a single underscore. Leading/trailing
       // underscores are trimmed.
       //
       // Examples:
       //   "Slice of Life" -> "slice_of_life"
       //   "Sci-Fi"        -> "sci_fi"
       //   "A & B"         -> "a_b"
       //   ""              -> ""
       //   "  Action  "    -> "action"
       func slugifyTagName(name string) string {
           lower := strings.ToLower(strings.TrimSpace(name))
           re := regexp.MustCompile(`[^a-z0-9]+`)
           slug := re.ReplaceAllString(lower, "_")
           slug = strings.Trim(slug, "_")
           return slug
       }

       // SlugifyTagName is the exported wrapper for slugifyTagName, used by
       // the Wave-2 backfill to compute domain.Tag.ID consistently.
       func SlugifyTagName(name string) string { return slugifyTagName(name) }
       ```
