Canonical service entrypoint pattern (catalog-api main.go reference):
```go
// services/catalog/cmd/catalog-api/main.go (simplified shape)
func main() {
  log := logger.Default()
  defer func() { _ = log.Sync() }()
  cfg, err := config.Load()  // panics fatal on error
  // ... business init (DB, redis, repos, services)
  router := transport.NewRouter(handlers..., cfg, log, metricsCollector)
  srv := &http.Server{Addr: cfg.Server.Address(), Handler: router, ReadTimeout: 30s, WriteTimeout: 30s, IdleTimeout: 60s}
  go srv.ListenAndServe()
  quit := make(chan os.Signal, 1)
  signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
  <-quit
  ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
  defer cancel()
  srv.Shutdown(ctx)
}
```

Canonical router pattern (catalog router.go reference):
```go
func NewRouter(/* handlers */, cfg *config.Config, log *logger.Logger, mc *metrics.Collector) http.Handler {
  r := chi.NewRouter()
  r.Use(middleware.RequestID)
  r.Use(mc.Middleware)
  r.Use(httputil.RequestLogger(log))
  r.Use(httputil.Recoverer(log))
  r.Use(httputil.CORS([]string{"*"}))
  r.Use(middleware.RealIP)
  r.Get("/health", func(w http.ResponseWriter, r *http.Request) {
    httputil.OK(w, map[string]string{"status":"ok"})
  })
  r.Get("/metrics", func(w http.ResponseWriter, r *http.Request) {
    metrics.Handler().ServeHTTP(w, r)
  })
  return r
}
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-01 | Information Disclosure | scraper /metrics + /health endpoints | accept | Bound to 127.0.0.1 host port; never routed through gateway. Internal-only API. |
| T-15-02 | Denial of Service | unbounded /metrics scrape | accept | Prometheus inside the docker network is the only configured scraper; no outside attack vector while host port is loopback-only. |
| T-15-03 | Spoofing | Unauthenticated calls from catalog → scraper | accept | Internal docker network; trust boundary is the docker bridge. Adding auth between two trusted internal services is overhead without threat reduction for a self-hosted 10-user deployment. |
</threat_model>
