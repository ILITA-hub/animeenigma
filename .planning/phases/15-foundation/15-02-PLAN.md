<objective>
Land the core domain types and interfaces that every provider, the orchestrator, and the HTTP handler layer will implement against. Five concerns:

1. **Provider interface + DTOs** — the contract providers implement. Stream DTO has no `iframe_url` field (type-level enforcement of D-DEC §2.8).
2. **Sentinel errors** — `ErrNotFound`, `ErrProviderDown`, `ErrExtractFailed` driving orchestrator failover semantics.
3. **EmbedExtractor interface + registry** — adding a new embed family is one registry entry.
4. **BaseHTTPClient** — `retryablehttp` + per-host `rate.Limiter` + `cookiejar`, 10s timeout, exponential backoff. Every provider in Phases 16-19 uses this; the type ships here so it cannot be hand-rolled later (D-DEC §5.3, SCRAPER-NF-01, SCRAPER-NF-03).
5. **CI lint** — a normal go test that parses `services/scraper/go.mod` and fails the build if any forbidden module path appears (D-DEC §5 Rejected, SCRAPER-FOUND-09).

**Purpose:** Wire the type-level guards in BEFORE any provider can possibly violate them. Once these tests are green and in CI, regressing them requires a deliberate PR that fails CI.

**Output:** Domain package with five files + tests + the lint test in a `golint` subpackage. No business logic. No HTTP server. No orchestrator yet (plan 03).
</objective>

The Provider interface shape (from ARCHITECTURE.md §2):
```go
package domain

type Category string
const (
    CategorySub Category = "sub"
    CategoryDub Category = "dub"
    CategoryRaw Category = "raw"
)

type AnimeRef struct {
    AnimeID     string // catalog UUID
    ShikimoriID string // == MAL ID
    AniListID   string
    Title       string
    Year        int
}

type Episode struct {
    ID       string `json:"id"`
    Number   int    `json:"number"`
    Title    string `json:"title"`
    IsFiller bool   `json:"is_filler"`
}

type Server struct {
    ID   string `json:"id"`
    Name string `json:"name"`
}

type Source struct {
    URL     string `json:"url"`
    Type    string `json:"type"`     // "hls" / "mp4"
    Quality string `json:"quality,omitempty"`
}

type Track struct {
    File    string `json:"file"`
    Label   string `json:"label,omitempty"`
    Kind    string `json:"kind"`     // "captions" / "subtitles"
    Default bool   `json:"default,omitempty"`
}

// Stream — CRITICAL: NO iframe_url field. Type-level enforcement.
type Stream struct {
    Sources []Source          `json:"sources"`
    Tracks  []Track           `json:"tracks,omitempty"`
    Intro   *TimeRange        `json:"intro,omitempty"`
    Outro   *TimeRange        `json:"outro,omitempty"`
    Headers map[string]string `json:"headers,omitempty"`  // e.g. Referer for proxy
    // NEVER add IframeURL string `json:"iframe_url"`. The compile-time test in provider_test.go forbids it.
}

type TimeRange struct {
    Start int `json:"start"`
    End   int `json:"end"`
}

type StageHealth struct {
    Up        bool      `json:"up"`
    LastOK    time.Time `json:"last_ok"`
    LastErr   string    `json:"last_err,omitempty"`
}

type Health struct {
    Provider string                 `json:"provider"`
    Stages   map[string]StageHealth `json:"stages"`
}

type Provider interface {
    Name() string
    FindID(ctx context.Context, ref AnimeRef) (string, error)
    ListEpisodes(ctx context.Context, providerID string) ([]Episode, error)
    ListServers(ctx context.Context, providerID, episodeID string) ([]Server, error)
    GetStream(ctx context.Context, providerID, episodeID, serverID string, category Category) (*Stream, error)
    HealthCheck(ctx context.Context) Health
}
```
