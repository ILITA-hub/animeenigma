<objective>
Implement the Kwik embed extractor — a self-contained in-process JS unpacker that decodes AnimePahe's Dean-Edwards-packed HLS m3u8 URLs using `github.com/dop251/goja`. The extractor registers as the second `domain.EmbedExtractor` in the registry (after Megacloud) so the AnimePahe provider in Plan 16-03 can route Kwik URLs to it transparently via `registry.Find(embedURL)`.

Purpose: SCRAPER-PAHE-03 requires HLS m3u8 URLs at 480p/720p/1080p resolved via kwik.cx + dop251/goja. The CONTEXT.md locks an in-process goja design (NOT a sidecar like Megacloud) because the Kwik unpacker is a self-contained packer with no DOM / browser APIs. Per RESEARCH.md Pitfalls 2 + 3, goja runtimes must be fresh per call and interrupted from a separate goroutine.

Output: A working kwik.go + kwik_test.go pair that unpacks the golden fixture from Plan 16-01 deterministically with zero network calls and zero shared state.
</objective>

// Host match pattern Kwik must mirror — host equality OR strict subdomain.
// SSRF guard: "evilmegacloud.tv" does NOT match "megacloud.tv".
func (c *MegacloudClient) Matches(embedURL string) bool {
    u, err := url.Parse(embedURL)
    if err != nil { return false }
    host := strings.ToLower(u.Hostname())
    if host == "" { return false }
    for _, known := range megacloudKnownHosts {
        if host == known || strings.HasSuffix(host, "."+known) { return true }
    }
    return false
}

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller (provider) → KwikExtractor.Extract(embedURL) | embedURL is caller-controlled but originates from upstream scraping in Plan 16-03; treat as untrusted. |
| KwikExtractor → upstream kwik.cx | Untrusted body; cap reads at 2 MiB; regex parse with bounded backtracking. |
| KwikExtractor → goja runtime | Untrusted JS (the Dean-Edwards packer expression). Fresh runtime per call; 5s interrupt; no native bindings; no fs/net/process exposed by default. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-02-01 | Tampering / Info disclosure (SSRF) | Matches() — attacker-controlled embedURL like `kwik.cx.attacker.com/e/abc` | mitigate | Strict host equality + HasSuffix("."+known); explicit test `TestKwik_Matches_RejectsSubdomainImposters` blocks regressions. Mirrors megacloud.go:86-101 pattern. |
| T-16-02-02 | Tampering / Elevation | goja sandbox escape | mitigate | goja's default Runtime exposes NO `fs`, `net`, `os`, `process`, `require`. Each Extract() constructs a fresh `goja.New()` — never reuses, never adds bindings, never exposes Go funcs to JS. |
| T-16-02-03 | Denial of service | Hostile upstream returns infinite-loop JS or 10 GiB response | mitigate | (a) `io.LimitReader(resp.Body, 2<<20)` caps body at 2 MiB. (b) `vm.Interrupt()` from a separate goroutine after 5s default timeout. (c) `ctx.Done()` propagation via same Interrupt path. |
| T-16-02-04 | DoS via data race | Concurrent Extract() calls corrupt a shared goja Runtime | mitigate | Fresh `goja.New()` per Extract call (RESEARCH.md Pitfall 2). Never cache the runtime. Pool/reuse strategies explicitly rejected. |
| T-16-02-05 | Tampering | Stream URL points to attacker host | mitigate | Downstream defense: `libs/videoutils/proxy.go::HLSProxyAllowedDomains` only permits whitelisted hosts. Plan 16-01 locks AnimePahe's CDN hosts in that allowlist. |
| T-16-02-06 | Repudiation | Silent extraction failures | mitigate | All error paths wrap with WrapExtractFailed (errors.Is matches domain.ErrExtractFailed); orchestrator failover loop in Plan 16-05 logs per-provider error. |
