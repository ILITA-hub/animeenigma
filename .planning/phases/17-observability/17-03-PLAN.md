<objective>
Ship the admin debug endpoint promised by SCRAPER-OBS-05: a JWT-admin-gated `GET /api/admin/scraper/health` that returns the per-provider per-stage health snapshot enriched with `last_ok` timestamps and truncated `last_err` excerpts from the in-memory health cache (Plan 17-01).

Purpose: Operators need a single curl-able endpoint for incident response — "is the provider dead or just slow?" should not require ssh'ing into the scraper container. This plan extends the scraper handler with an admin variant, mounts it under `/scraper/health/admin`, and threads the gateway routing (config + proxy URL + handler + router) so that gateway JWT+Admin middleware sits in front. The defense model from D6 is gateway-only auth (scraper binds to 127.0.0.1 inside the docker network — A5 assumption documented).

   ```go
   // GetAdminHealth handles GET /scraper/health/admin. Returns the orchestrator's
   // public HealthSnapshot alongside the in-memory cache's enriched AdminSnapshot
   // (which includes LastOK timestamps and truncated LastErr excerpts).
   //
   // Auth: enforced by the gateway via JWTValidationMiddleware + AdminRoleMiddleware.
   // This handler trusts the gateway gate (A5 assumption — scraper binds to
   // 127.0.0.1 inside the docker network, so no defense-in-depth auth here).
   //
   // Defense-in-depth: re-truncates any LastErr that exceeded MaxLastErrChars
   // (Plan 17-01's probe already truncates, but new code paths might forget — P-05).
   func (h *ScraperHandler) GetAdminHealth(w http.ResponseWriter, r *http.Request) {
       public := h.svc.HealthSnapshot(r.Context())

       var enriched map[string]health.ProviderHealth
       if h.cache != nil {
           enriched = h.cache.AdminSnapshot()
           // Defense-in-depth truncation. The cache layer already truncates
           // before Update, but a future caller could bypass that path.
           for prov, ph := range enriched {
               for st, ss := range ph.Stages {
                   if len(ss.LastErr) > health.MaxLastErrChars {
                       ss.LastErr = ss.LastErr[:health.MaxLastErrChars]
                       ph.Stages[st] = ss
                   }
               }
               enriched[prov] = ph
           }
       } else {
           enriched = map[string]health.ProviderHealth{}
       }

       httputil.OK(w, map[string]any{
           "providers":    public,
           "admin":        enriched,
           "generated_at": time.Now().UTC().Format(time.RFC3339),
       })
   }
   ```

   Add the `"time"` import if not already present.
