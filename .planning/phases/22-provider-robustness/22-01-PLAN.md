<objective>
Extend the StreamHG and Earnvids embed extractors so they return BOTH the `hls2` signed `.m3u8` AND the `hls3` unsigned `.txt` URLs from the unpacked packed-JS body — as separate entries in `Stream.Sources`, ordered hls2-first. Wire `gogoanime.coldPathGated` (Phase 21 cold-path) to actually iterate the multi-source list via the playability gate so the second URL family is tried when the first fails (signed-URL expired / CDN 403 / geo-block).

Purpose: when a single CDN behind a server fails (signed-URL expired, 403, geo-block), the orchestrator transparently tries the server's secondary URL family BEFORE giving up on the server. Distinct from Phase 21's "the server is dead" — this catches "the regex still works but the URL doesn't".

Output:
- Shared multi-URL extractor `extractAllPlayableURLs` in `packed_common.go` capturing both `.m3u8` and `.txt` URLs from the unpacked dictionary, deduped, hls2-first ordering.
- `streamhg.go` + `earnvids.go` updated to use the new helper; return `[]domain.Source` of length 1-2 (gracefully handles regressions where only one URL is present).
- `coldPathGated`'s `attemptOne` iterates `s.Sources` rather than indexing `[0]` — without this change, the second extracted URL is dead code.
- Updated golden fixtures `streamhg_packed.html` + `earnvids_packed.html` re-captured from the PoC 2026-05-13 unpacks so both `hls2` and `hls3` URLs are present (or augmented if existing goldens contain only one).
- Unit tests asserting: 2-entry result on multi-URL goldens, 1-entry result when packed JS exposes only one URL, hls2-first ordering, per-Source `Headers` includes upstream `Referer`.
</objective>

      func extractAllPlayableURLs(unpacked string) []domain.Source {
          seen := make(map[string]struct{}, 4)
          var out []domain.Source
          add := func(url string) {
              if _, dup := seen[url]; dup { return }
              seen[url] = struct{}{}
              out = append(out, domain.Source{URL: url, Type: "hls"})
          }
          for _, m := range hls2Regex.FindAllStringSubmatch(unpacked, -1) { add(m[1]) }
          for _, m := range hls3Regex.FindAllStringSubmatch(unpacked, -1) { add(m[1]) }
          // Fallback pass — picks up rotated key names (streamA/B). Limit 5 entries to avoid
          // a hostile body inflating the source list arbitrarily.
          for i, m := range genericPlayableRegex.FindAllStringSubmatch(unpacked, -1) {
              if i >= 5 { break }
              add(m[1])
          }
          return out
      }
