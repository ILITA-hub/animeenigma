## Threat Model

ASVS Level 1; block-on=high.

| Threat ID | STRIDE | Component | Disposition | Mitigation |
|-----------|--------|-----------|-------------|------------|
| T-27-02-01 | Info disclosure | `Provider.ListServers` — Kwik URL reaching the goja extractor with a non-http(s) scheme (path-traversal-style `kwik://path/../../../`) | mitigate (LOW) | Preserve the existing WR-05 scheme check in `client.go::ListServers` — `pu.Scheme != "http" && pu.Scheme != "https"` rejects the entry. Already covered by `TestProvider_ListServers_SchemeReject` in `client_test.go`; rewiring the test to the resolver mock preserves the assertion. |
| T-27-02-02 | Tampering | `resolverClient` calling an attacker-controlled `SCRAPER_ANIMEPAHE_RESOLVER_URL` | mitigate (LOW) | `config.go` validates the URL has a scheme + host at boot (mirrors the existing `ANIMEPAHE_BASE_URL` validation). The variable is only operator-set in compose; not user-influenceable. |
| T-27-02-03 | DoS | Unbounded resolver response body | mitigate (LOW) | Preserve existing `maxBodyAPI` (4 MiB) + `maxBodyHTML` (2 MiB) `io.LimitReader` caps in `client.go` and replicate them in `resolver.go`. |

This plan does NOT own the SSRF / sandbox / OOM / cookie-leak / stealth-plugin threats — those are owned by Plan 27-01.
