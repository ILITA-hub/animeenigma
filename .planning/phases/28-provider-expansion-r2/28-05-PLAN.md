<objective>
Implement the 9anime.me.uk provider as a new self-contained package under `services/scraper/internal/providers/nineanime/`. FindID via WP REST API title-fuzzy with Shikimori/MAL metadata tie-breakers (year, episode count). ListEpisodes scrapes the series page for `<a class="ep-item" data-number>` elements (no URL guessing per Pitfall 5). Allowlist updated for `my.1anime.site` per D7.

Failover slot 6 means this provider is probed only when the 5 prior providers fail. When 9anime breaks (~6-month half-life), operator mitigation is `SCRAPER_DEGRADED_PROVIDERS=nineanime`.

Output: New `nineanime` package + config block + main.go registration + 1 new allowlist entry.
