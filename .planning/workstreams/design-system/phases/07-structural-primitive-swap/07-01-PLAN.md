```
Props: variant, size, type ('button'|'submit'|'reset', default 'button'), href, disabled,
       loading, fullWidth, class (HTMLAttributes['class'] — merged via cn())
- Renders <a> when href set, else <button>. Forwards arbitrary attrs (data-*, aria-*, title,
  :aria-pressed, :aria-expanded) onto the root element via reka-ui <Primitive> fallthrough.

  utilities through `class` (e.g. class="mt-4 w-full").
```
