<objective>
Wire the spotlight aggregator into the catalog HTTP surface:

1. Add `SpotlightEnabled bool` to catalog config + `getEnvBool` helper.
2. Create `services/catalog/internal/handler/spotlight.go` — chi handler that
   short-circuits 404 when the flag is off, calls the aggregator, and writes
   the bare `{cards, generated_at}` envelope directly via `json.NewEncoder`
   (DELIBERATE DIVERGENCE 3: NOT `httputil.OK` which would wrap in
   `{success,data}`).
3. Extend `transport.NewRouter` to take a `*handler.SpotlightHandler` arg and
   register `r.Get("/home/spotlight", spotlightHandler.Get)` inside the `/api`
   group as a public sibling of `/api/anime/news`.
4. Wire everything in `services/catalog/cmd/catalog-api/main.go`: construct
   the web client → 4 resolvers → aggregator → handler → pass to router.

Purpose: Phase 1 demo. After this plan + Plan 05 (gateway), a developer can
`curl http://localhost:8000/api/home/spotlight` and see real cards.

DELIBERATE DIVERGENCE 3 (non-negotiable, baked into acceptance):
- Handler MUST use `json.NewEncoder(w).Encode(resp)` directly with manual
  Content-Type header. Reason: the design doc §4.1 specifies a bare
  `{cards, generated_at}` envelope. `httputil.OK` wraps in
  `{success:true, data:{...}}` which the frontend HSB-FE-02 would still
  parse as truthy but would surface the wrong cards path.
- Feature-flag-off short-circuit MUST use `w.WriteHeader(http.StatusNotFound); return` directly with NO body.
  Reason: `httputil.NotFound` emits `{success:false, error:{...}}` JSON which the frontend HSB-FE-02 would parse as truthy.

real outward sign of the feature).

+ router + main.go wiring are mechanical but easy to break).
MVQ = Griffin 90%/85% (HTTP surface assembly; slop risk is using

       // hero-spotlight, v1.0 Phase 1). Public, no auth. Returns a bare
       // {cards, generated_at} JSON envelope — NOT the libs/httputil
       // {success, data} envelope. This divergence is deliberate; see
       // PATTERNS.md §"HTTP response pattern (DIVERGENCE from httputil.OK)".

           enabled bool
           log     *logger.Logger

    2.3. Get handler:
       ```go
       // Get handles GET /api/home/spotlight. Returns 200 with the
       // bare envelope on success (including partial success — at
       // least one card resolved). Returns 404 (no body) when the
       // feature flag is off. Returns 500 with an empty-cards body
       // only on a catastrophic aggregator error.
       func (h *SpotlightHandler) Get(w http.ResponseWriter, r *http.Request) {
           // Feature flag — HSB-BE-07. Bare 404, no body. NOT httputil.NotFound.
           if !h.enabled {
               w.WriteHeader(http.StatusNotFound)
               return
           }

           started := time.Now()

           // Phase 1 — endpoint is public; Phase 3 will add optional-auth
           // and pass a real *string userID. Tolerate Authorization header
           // being present (do not 401, do not parse).
           h.log.Infow("spotlight.request", "user", "anon")

           ctx, cancel := context.WithTimeout(r.Context(), 2*time.Second)
           defer cancel()

           resp, err := h.agg.Resolve(ctx, nil)
           if err != nil {
               h.log.Errorw("spotlight.aggregator_failed", "error", err)
               w.Header().Set("Content-Type", "application/json")
               w.WriteHeader(http.StatusInternalServerError)
               _ = json.NewEncoder(w).Encode(spotlight.Response{
                   Cards:       []spotlight.Card{},
                   GeneratedAt: time.Now().UTC().Format(time.RFC3339),
               })
               return
           }

           h.log.Infow("spotlight.aggregated",
               "cards_returned", len(resp.Cards),
               "ms_total", time.Since(started).Milliseconds(),
           )

           w.Header().Set("Content-Type", "application/json")
           w.WriteHeader(http.StatusOK)
           if err := json.NewEncoder(w).Encode(resp); err != nil {
               h.log.Errorw("spotlight.encode_failed", "error", err)
           }
       }
       ```

    CRITICAL constraints — bake into the code AND the acceptance grep gates:
    - DO NOT import `libs/httputil` in this file.
    - DO NOT call `httputil.OK`, `httputil.JSON`, `httputil.Error`, or `httputil.NotFound`.
    - The flag-off short-circuit MUST be exactly `w.WriteHeader(http.StatusNotFound); return` — no body.
    - The success path MUST be `json.NewEncoder(w).Encode(resp)`.
