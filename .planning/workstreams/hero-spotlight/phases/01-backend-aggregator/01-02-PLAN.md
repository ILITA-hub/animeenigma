<objective>
Implement the four Phase-1 card resolvers + the web client that backs
`latest_news`. Each resolver implements the `spotlight.Resolver` interface
defined in Plan 01: `Type() string` + `Resolve(ctx, userID *string) (*Card, error)`.

Purpose: Lock down all four data sources behind a uniform contract so Plan 03
can fan-out across them concurrently without knowing implementation details.
Resolvers are independent — they can be implemented in parallel (and the
package compiles after any subset lands).

Output: 4 resolver structs + 1 web client + 5 test files. Each resolver
returns a typed `*Card` with the correct discriminator and cached data;
returns `(nil, nil)` when ineligible (zero metrics, empty top-rated, etc).

DELIBERATE DIVERGENCE 1 (non-negotiable, baked into acceptance):
- Use manual `cache.Get` + `errors.Is(err, cache.ErrNotFound)` + `cache.Set`,
  NOT `cache.GetOrSet`. Reason: `GetOrSet` caches nil/empty zero values,
  baking a 24h "no data" cache that breaks eligibility recovery.

UXΔ = +1 (Better — surfaces real catalog data into a new hero block; user
sees fresh content daily).
CDI = 0.10 * 8 (10 new files: 4 resolvers + 4 resolver tests + 1 web client
+ 1 web client test, Effort=8 because two of them — anime_of_day/random_tail
— share Search() repo wiring with a known gotcha; latest_news involves an
HTTP fetch with httptest stub; platform_stats has the GORM count + nil-metric
eligibility branch).
MVQ = Griffin 90%/85% (concurrent-resolvable, deterministic, cacheable —
fits the "spotlight" theme; slop risk in latest_news flattening logic if
changelog.json shape isn't read carefully).
</objective>

       ```
       key := "spotlight:anime_of_day:" + spotlight.DateKeyUTC(time.Now())

       var cached spotlight.AnimeOfDayData
       if err := r.cache.Get(ctx, key, &cached); err == nil {
           return &spotlight.Card{Type: r.Type(), Data: cached}, nil
       } else if !errors.Is(err, cache.ErrNotFound) {
           r.log.Warnw("spotlight.cache_get_failed", "type", r.Type(), "error", err)
       }

       sm := 8.0
       items, _, err := r.repo.Search(ctx, domain.SearchFilters{
           Sort: "score", Order: "desc",
           ScoreMin: &sm, Page: 1, PageSize: 200,
       })
       if err != nil {
           return nil, fmt.Errorf("anime_of_day: repo search: %w", err)
       }
       if len(items) == 0 {
           return nil, nil // eligible=false
       }
       picked := items[spotlight.DateSeedUTC(time.Now())%len(items)]
       data := spotlight.AnimeOfDayData{Anime: *picked}

       if err := r.cache.Set(ctx, key, data, 24*time.Hour); err != nil {
           r.log.Warnw("spotlight.cache_set_failed", "type", r.Type(), "error", err)
       }
       return &spotlight.Card{Type: r.Type(), Data: data}, nil
       ```
