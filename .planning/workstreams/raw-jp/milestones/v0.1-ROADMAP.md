# Roadmap: v0.1 Raw Provider MVP

**Workstream:** raw-jp
**Milestone:** v0.1 (active)
**Created:** 2026-05-18
**Source design doc:** `/data/animeenigma/docs/superpowers/specs/2026-05-18-raw-jp-provider-design.md`
**Requirements:** `v0.1-REQUIREMENTS.md`

## Goal

Ship a working RAW Japanese audio player with multi-language subtitle aggregation. New `'raw'` provider chip in the language switcher. "Other subs" button opens a panel listing every available subtitle file across providers, grouped by language. Behind feature flag for soft rollout via `ui_audit_bot` first.

## Phases

### Phase 1: AllAnime Parser

**Goal:** Implement `services/catalog/internal/parser/allanime/` â€” GraphQL client against `api.allanime.day` with `translationType: raw`. Persisted-query SHA hashes in env config. Rotating-domain support. Catalog endpoints `/api/anime/{id}/raw/episodes` and `/raw/stream?episode={n}&quality={q}`.

**Depends on:** Nothing â€” additive backend module. Can run in parallel with Phase 2.
**Requirements:** RAW-01, RAW-02, RAW-NF-01
**SPEC:** `v0.1-phases/01-allanime-parser/01-SPEC.md`
**Touches:**
- `services/catalog/internal/parser/allanime/{client.go,queries.go,episodes.go,domains.go}` (new)
- `services/catalog/internal/handler/anime.go` (extend with new routes)
- `services/catalog/internal/service/raw_resolver.go` (new)
- `services/catalog/internal/domain/anime.go` (extend with raw source type)
- `docker/.env` (add `ALLANIME_QUERY_*_SHA` vars)

**Success criteria:**
1. `curl http://localhost:8081/api/anime/{shikimori_id}/raw/episodes` returns an episode list for a known AllAnime-covered title (e.g. Bocchi the Rock, MAL ID 52082) with non-empty `episodes` array.
2. `curl http://localhost:8081/api/anime/{shikimori_id}/raw/stream?episode=1` returns a valid HLS playlist URL.
3. Domain rotation works: with `allanime.day` simulated down, the parser falls through to `allmanga.to` and the request succeeds.
4. Stream URL cache hit on second call within 1 hour (verified via metrics or log).
5. Unit tests against recorded GraphQL fixtures pass. Integration tests gated on `INTEGRATION=1` hit the live API and pass.

### Phase 2: Subtitle Aggregator + Extended ID Mapping

**Goal:** New parser `services/catalog/internal/parser/opensubtitles/` against `api.opensubtitles.com/api/v1`. New aggregator service merges Jimaku + OpenSubtitles results, dedupes by content hash, groups by language. Extend `libs/idmapping/` to resolve IMDb/TMDB via Kitsu mappings.

**Depends on:** Nothing â€” additive. Can run in parallel with Phase 1.
**Requirements:** RAW-03, RAW-04, RAW-NF-01
**SPEC:** `v0.1-phases/02-subtitle-aggregator/02-SPEC.md`
**Touches:**
- `services/catalog/internal/parser/opensubtitles/client.go` (new)
- `services/catalog/internal/service/subs_aggregator.go` (new)
- `services/catalog/internal/handler/subtitles.go` (new)
- `libs/idmapping/kitsu.go` (new â€” IMDb/TMDB resolver via Kitsu)
- `libs/idmapping/cache.go` (extend to include IMDb/TMDB columns)
- `services/catalog/migrations/` or GORM AutoMigrate path â€” add `imdb_id` + `tmdb_id` to `animes`
- `docker/.env` (add `OPENSUBTITLES_API_KEY`, `OPENSUBTITLES_USER_AGENT`)

**Success criteria:**
1. `curl http://localhost:8081/api/anime/{id}/subtitles?lang=jp,ru,en` returns â‰Ą1 result in at least 2 of the 3 requested languages for a major title.
2. `curl http://localhost:8081/api/anime/{id}/subtitles/all` returns every track (no language filter), grouped by language code in the response.
3. Each result includes `provider`, `language`, `url`, `format` (`ass`|`srt`|`vtt`), `episode` (when known).
4. OpenSubtitles rate-limit (429) downgrades to Jimaku-only without surfacing 5xx to the client.
5. `animes.imdb_id` populated for at least one queried anime after the request completes.

### Phase 3: RawPlayer.vue + Other Subs Panel

**Goal:** Build the frontend player and "Other subs" modal. RawPlayer.vue is structurally close to HiAnimePlayer.vue (HLS.js + Video.js, SubtitleOverlay mounted at fullscreen target) but with the subtitle picker promoted to a primary toolbar control. OtherSubsPanel.vue is a modal listing every track grouped by language with provider attribution.

**Depends on:** Phase 1 + Phase 2 endpoints exist and respond.
**Requirements:** RAW-05, RAW-06
**SPEC:** `v0.1-phases/03-raw-player-frontend/03-SPEC.md`
**Touches:**
- `frontend/web/src/components/player/RawPlayer.vue` (new)
- `frontend/web/src/components/player/OtherSubsPanel.vue` (new)
- `frontend/web/src/components/player/SubtitleOverlay.vue` (reused unchanged)
- `frontend/web/src/utils/subtitle-parser.ts` (reused unchanged)
- `frontend/web/src/api/client.ts` (add raw endpoints + subtitle endpoints)
- `frontend/web/src/locales/{en,ru,ja}.json` (new strings for "Other subs", language labels)

**Success criteria:**
1. `RawPlayer.vue` renders a video element + subtitle overlay + primary subtitle picker dropdown that lists languages returned from `/subtitles`.
2. Default subtitle pick respects user's `preferred_subtitle_language` localStorage value, falling back to the first available track.
3. Clicking "Other subs" opens `OtherSubsPanel.vue` listing every track from `/subtitles/all` grouped by language with a provider chip on each row.
4. Selecting a track in the panel updates the SubtitleOverlay immediately without page reload.
5. Locale strings load correctly in all three project locales with no `[intlify]` warnings.

**Depends on:** Phase 3 components.
**Requirements:** RAW-07, RAW-08, RAW-NF-02
**SPEC:** `v0.1-phases/04-frontend-wiring/04-SPEC.md`
**Touches:**

- `frontend/web/e2e/raw-player.spec.ts` (new Playwright test)
- `frontend/web/public/changelog.json` (new entry)
- `frontend/web/src/locales/{en,ru,ja}.json` (language-group label "RAW JP")

**Success criteria:**
1. With `VITE_RAW_PROVIDER_ENABLED=true`, opening any anime detail page shows three language-group chips: RU, EN, RAW JP.
2. Selecting RAW JP loads `RawPlayer.vue` and the raw episode list.
3. Provider preference persists across reload (`preferred_raw_provider` localStorage key).
4. Playwright e2e against `ui_audit_bot` passes: login â†’ open known anime â†’ switch to RAW JP â†’ assert load â†’ assert subtitle picker populated â†’ assert "Other subs" panel opens.
