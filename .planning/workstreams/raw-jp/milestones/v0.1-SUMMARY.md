## What shipped

End-to-end "RAW JP" video provider serving original Japanese audio + a multi-language subtitle aggregator panel. Behind the `VITE_RAW_PROVIDER_ENABLED` build flag in production until the operator runbook for refreshing AllAnime persisted-query SHAs (see ISS-012) is exercised on production data.

### Phase 1 â€” AllAnime Parser
- `services/catalog/internal/parser/allanime/` â€” pure HTTP+JSON GraphQL client (`client.go`, `domains.go`, `queries.go`, `episodes.go`).
- Rotating domains `[allanime.day, allmanga.to, allanime.to]` with first-success caching + 5-min cooldown on failure.
- Persisted-query SHA hashes overridable via `ALLANIME_QUERY_{SEARCH,EPISODES,SOURCES}_SHA`.
- `services/catalog/internal/service/raw_resolver.go` â€” wraps the parser with 6h-episodes + 1h-stream cache, singleflight-deduped show-ID lookup, transport-vs-content error classification (transport failures â†’ 503, content rejects â†’ graceful empty).
- `services/catalog/internal/handler/raw.go` mounts `GET /api/anime/{id}/raw/episodes` and `/raw/stream?episode=N&quality=Q`.
- `domain.SourceTypeRaw`, `Anime.HasRaw` column, `AnimeRepository.SetHasRaw` for lazy backfill.
- 11 unit tests against httptest mocks.

### Phase 2 â€” Subtitle Aggregator + Extended ID Mapping

- `services/catalog/internal/service/subs_aggregator.go` â€” goroutine fan-out across Jimaku + OpenSubtitles, fail-soft per provider, lazy IMDb/TMDB backfill via Kitsu, 6h merged response cache.
- `services/catalog/internal/handler/subtitles.go` mounts `GET /api/anime/{id}/subtitles?lang=ja,en,ru&episode=N` and `/subtitles/all?episode=N`. Sets `X-Subtitle-Providers-Down: <csv>` header for observability.
- `domain.Anime.IMDbID` / `TMDBID` columns, `AnimeRepository.UpdateExternalIDs`.
- 12 unit tests.
