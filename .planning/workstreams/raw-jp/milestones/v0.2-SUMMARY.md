## What shipped

A self-hosted raw-JP video pipeline that turns admin-curated Nyaa/AnimeTosho releases into HLS streams on MinIO, transparently preferred over AllAnime by the existing `/raw/stream` endpoint when present, and falling back gracefully when not. End-to-end: search → BitTorrent download → ffmpeg HLS → MinIO → episodes endpoint → hybrid catalog resolver.

### Phase 1 — Library Service Scaffold
- New Go microservice `services/library/` (port 8089 — Phase 1 deviation from SPEC's 8087, see open items).
- Standard layout: `cmd/library-api/main.go`, `internal/{config,domain,handler,repo,service,transport,metrics,torrent,ffmpeg,minio,parser/{filename,nyaa,animetosho}}`, `migrations/`.
- `libs/database` + GORM auto-create of dedicated `library` Postgres DB.
- `libs/metrics` + `libs/logger` integration; `chi` router with the standard middleware chain.
- `services/gateway/internal/{config,handler,service,transport}` extended with `/api/library/*` reverse-proxy block.
- `docker/docker-compose.yml` `library` service (4 CPU / 4 GiB) + named volumes `library_torrents`, `library_minio_staging`.
- `Makefile` `redeploy/restart/logs-library` + extended `health` target.
- `CLAUDE.md` Service Ports + Gateway Routing tables updated.

### Phase 2 — Nyaa + AnimeTosho Search Clients
- `services/library/internal/parser/nyaa/` — RSS client (namespaced XML) with synthesized magnet URIs and table-driven size parser.
- `services/library/internal/parser/animetosho/` — JSON feed client with `show=mal` preferred path + query fallback.
- `services/library/internal/domain/release.go` — shared `Release{Title, Magnet, InfoHash, Uploader, Quality, SizeBytes, Source, MALID, FoundAt}` type.
- `services/library/internal/service/search.go` — `SearchAggregator` with `errgroup.SetLimit(2)`, fail-soft per provider, InfoHash dedupe preferring AnimeTosho, MAL-aware ranking.
- `services/library/internal/handler/search.go` — `GET /api/library/search?q=&mal_id=&limit=` admin-gated; response envelope `{releases, providers_down}`.
- Gateway admin gate added for `/api/library/*` non-`/health` routes, mirroring `/streaming/admin/*`.
- 17 unit tests across parsers + service.

### Phase 3 — Torrent Client + Job Queue + Metrics
- `services/library/internal/torrent/` — facade over `github.com/anacrolix/torrent`: `Add(ctx, magnet)`, `Progress`, `Cancel`, `Close`; single `*torrent.Client` per process.
- `services/library/migrations/001_library_jobs.sql` + `domain.Job` + `repo.Job` (Create/Get/List/Claim/UpdateProgress/UpdateStatus/Cancel/ResumeInterruptedDownloads). Claim uses `GORM clause.Locking{Strength:"UPDATE", Options:"SKIP LOCKED"}` inside a transaction.
- `services/library/internal/service/download_worker.go` — worker pool (default 2 via `LIBRARY_DOWNLOAD_WORKERS`); SIGTERM → ctx-aware shutdown + flip in-flight back to queued.
- `services/library/internal/service/disk_guard.go` — `unix.Statfs` polling every 30s; HTTP 507 enqueue rejection when `freePct < LIBRARY_DISK_FREE_MIN_PCT`.
- Stall detection — `lastNonZeroPeerAt`; ages job to `failed` past `LIBRARY_TORRENT_STALL_TIMEOUT`.
- `services/library/internal/metrics/library_metrics.go` — six collectors (`library_jobs_total{status}`, `library_download_bytes_total`, `library_active_torrents`, `library_disk_free_bytes`, `library_enqueue_rejected_total{reason}`, `library_torrent_seed_count`).
- `services/library/internal/handler/jobs.go` — POST/GET/DELETE on `/api/library/jobs`; admin-gated; magnet validation; disk-full + invalid-magnet rejection paths.
- `infra/grafana/dashboards/library.json` — first library dashboard, 7 panels.
- Live smoke downloaded Big Buck Bunny (~268 MB) inside the container; cancel + disk-full + metrics paths all confirmed.

### Phase 4 — ffmpeg HLS Transcoder + MinIO Writer
- `services/library/internal/ffmpeg/transcoder.go` — `os/exec` wrapper with ffprobe pre-flight (duration + source bitrate), 2 KB bounded stderr ring buffer, locked argv for libx264/aac/HLS/6s/VOD.
- `services/library/internal/minio/writer.go` — `minio-go/v7` client; idempotent `EnsureBucket(raw-library)` (swallows BucketAlreadyOwnedByYou); concurrent segment upload via errgroup (limit 8); playlist uploaded LAST behind a sync barrier; per-extension content-types.
- `services/library/migrations/002_library_episodes.sql` + `003_library_filename_patterns.sql` (idempotent seed for Ohys-Raws, SubsPlease, Erai-raws, Leopard-Raws, ARC-Raws).
- `services/library/internal/parser/filename/detector.go` — uploader-keyed regex catalogue + generic `- (\d+)` fallback emitting `library_filename_detect_fallback_total{uploader}`.
- `services/library/internal/service/encoder_worker.go` — separate worker pool consuming `encoding` jobs; transcode → upload → `library_episodes` insert (when shikimori_id known) → done; `pending/{job_id}/{ep}/` path when shikimori_id is NULL.
- `services/library/internal/handler/episodes.go` — `GET /api/library/episodes/{shikimori_id}/{episode}` admin-gated; 200 with `{minio_url, duration_sec, size_bytes}` or 404.
- `services/library/Dockerfile` extended with `apt-get install -y ffmpeg` in runtime; binaries at `/usr/bin/ffmpeg` + `/usr/bin/ffprobe`.
- Four new metrics: `library_encode_duration_seconds`, `library_upload_bytes_total`, `library_filename_detect_fallback_total{uploader}`, `library_encode_failures_total{reason}`.
- Synthetic encode smoke (10s testsrc → MinIO → `library_episodes` row → episodes endpoint roundtrip) verified.

### Phase 5 — RawLibrary.vue Admin UI
- `frontend/web/src/views/admin/RawLibrary.vue` — three-section view (stats strip, search panel, jobs panel) with 30s + 5s polling, debounced (300ms) search, provider chips (cyan AnimeTosho / purple Nyaa) via `Badge.vue` info/destructive variants.
- Sub-panels in the jobs section: failed jobs with error_text + Retry button; pending-link panel with debounced anime search dropdown.
- `frontend/web/src/api/client.ts` — `adminLibraryApi` (search, listJobs, getJob, createJob, cancelJob, linkJob, retryJob, healthExtended).
- `frontend/web/src/types/library.ts` — Release / Job / JobStatus / Episode / LibraryHealth / CreateJobPayload.
- Router entry `/admin/raw-library` with `requiresAuth + requiresAdmin` (existing global guard).
- i18n keys `player.adminLibrary.*` in en / ru / ja.
- Backend additions: `GET /api/library/health/extended` (disk + active-torrent summary); `PATCH /api/library/jobs/{id}` (NULL → shikimori_id transition triggers MinIO server-side `CopyObject` + `library_episodes` insert + old-path delete); `POST /api/library/jobs/{id}/retry`.
- `services/library/internal/minio/writer.go` extended with `Move(src, dst)` helper.
- `services/library/internal/repo/job.go` extended with `UpdateShikimoriID` + `Retry`.
- Playwright e2e `frontend/web/e2e/raw-library-admin.spec.ts` (promotes ui_audit_bot to admin in beforeAll, reverts after).
- All backend additions unit-tested.

### Phase 6 — Hybrid Resolver
- `services/catalog/internal/parser/library/client.go` — thin client `GetEpisode(shikimori_id, episode)` against `http://library:8089/api/library/episodes/{shikimori_id}/{episode}` with 2s timeout; 404 → `(nil, nil)`; 5xx/timeout → `(nil, error)`.
- `services/catalog/internal/service/raw_resolver.go` — library-first branch with Redis `raw:source-decision:{animeID}:{episode}` cache (1h TTL). Returns `source: "library"` + MinIO URL when present; otherwise falls through to v0.1 AllAnime path with `source: "allanime"`.
- `services/catalog/internal/handler/internal_cache.go` — new internal route `POST /internal/cache/invalidate/raw/{shikimoriId}` (lives outside `/api/*`, gateway doesn't proxy `/internal/*`). Translates shikimori_id → animeID and DELs the three matching key namespaces.
- `services/library/internal/service/encoder_worker.go` extended with best-effort webhook fire to `http://catalog:8081/internal/cache/invalidate/raw/{shikimori_id}` after every successful `done`; new `library_cache_invalidation_total{result=ok|fail}` metric.
- Live smoke: library hit → `source:"library"`; library stopped → fallback in 16ms (well under 2.5s SLA); webhook end-to-end verified.
- v0.1 frontend contract preserved (`RawStream.Source` field is additive; `types/raw.ts` ignores unknown).
