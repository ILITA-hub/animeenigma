The plan-as-written listed these direct backend deps:

> `gorilla/websocket`, `redis/go-redis/v9`, `golang.org/x/time/rate`, `google/uuid`, `prometheus/client_golang`, project libs (`logger, metrics, jwt, errors`)

The plan's explicit instruction was: *"`grep -E '^\s+github\.com|^\s+golang\.org' services/watch-together/go.mod | sort` — confirm the list in CLAUDE.md matches reality. Add/remove dep names accordingly."* — so this is the prescribed deviation path, not a Rule 1/2/3 trip.

Corrections written into CLAUDE.md:
- **Added** `github.com/go-chi/chi/v5` (HTTP router — required, missing from plan list)
- **Added** `github.com/golang-jwt/jwt/v5` (direct require — plan claimed a `libs/jwt` wrapper, but no such project lib is referenced from this `go.mod`)
- **Added** `github.com/prometheus/client_model` (direct require alongside `client_golang`)
