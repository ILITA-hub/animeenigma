## Cross-view consistency

- **Header z-index and drawer**: only one pattern (navbar z-50, drawer inline below), and it loses a stacking fight with any `z-[50+]` sibling in the page content (Home is the only current example, hence UA-056). If other views start adopting high-z-index wrappers, the same bug will appear there.
- **Dropdown background**: Home, Browse, and Navbar all use the post-`bd45b4a` opaque stack (`bg-gray-950/95 backdrop-blur-xl border shadow-2xl`). No residual transparent dropdowns found.
- **Footer contrast fix (Batch B)**: holds across all 7 views on mobile.
- **Document titles (Batch F)**: all static view titles correctly localized. Dynamic (anime detail) title is the one remaining gap (UA-051).
- **`text-white/40` residual**: concentrated on Anime detail (UA-052) — same pattern that was partially fixed in Batch B but scoped out. Other views now don't leak this pattern at all.
- **i18n hygiene**: two English leak-throughs found — `aria-label="Open menu"/"Close menu"` (UA-043), and the Anime error state "Failed to fetch anime" (UA-050).

## Audit notes

- **Token discipline**: used filtered DOM probes via `javascript_tool` rather than full `read_page` dumps. Per-view findings written immediately to disk, not batched at the end.
- **Viewport**: Chrome's window minimum clamps at ~500px inner width on this host; chose 500×723 as the closest-available mobile-portrait size. All reported findings reproduce at that width and should also reproduce at 390–414px (iPhone portrait) since the same Tailwind breakpoints fire (< `sm: 640`).
- **Probe repro**: every finding was re-probed at least once (axe rerun, DOM re-inspect, screenshot); transient results filtered out.
- **Scope filtered**: mobile-specific. Desktop 2026-04-17 re-audit already covered desktop state; this audit does NOT re-probe desktop.
- **Deferred / out-of-scope**: full auth-view re-audit, touch-driving L2/L3 (watchlist status change, sort by score on mobile), game-room join flow (G1), Kodik iframe subtitle flow.
- **Bot detection**: none tripped. All probes completed on prod.
