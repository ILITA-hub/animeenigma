# Active plans

This directory is an execution surface, not a historical archive. There are no
active cross-cutting plans after the 2026-07-15 review. Feature-level
implementation plans belong in [`../superpowers/plans/`](../superpowers/plans/);
design contracts live in [`../superpowers/specs/`](../superpowers/specs/).

When work ships, is abandoned, or is superseded, delete its plan after folding
durable architecture and operational details into a reference or runbook. Git
history retains the checklist and decision trail.
