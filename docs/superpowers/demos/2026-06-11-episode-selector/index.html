③ V2 на длинных тайтлах — 3 суб-варианта

У каждого плеера переключатель датасета: 12 (обычный сезон) / 112 (длинный) / 1042 (One Piece-класс). Лента всегда открывается с текущей серией (EP 4) по центру.

За
  • Лента = «что смотреть дальше» (названия, прогресс), грид = «навигация по архиву» — каждый режим делает своё
  • Грид внутри шита скроллится вертикально: 1042 ячейки обозримы за 2-3 прокрутки
  • Клик по ячейке грида → переключение в ленту с центром на выбранной серии (мостик между режимами)

  • Грид-режим повышает шит до ~200px — больше перекрытие видео
Edge cases: 12 серий → toggle и jump скрываются (просто лента); грид помнит выбранный режим per-session; на тач грид-ячейки 44px-таргеты.

  • Просмотр «через границу» (50→51) требует смены диапазона
Edge cases: 12 серий → диапазоны скрыты; ≤6 диапазонов → чипы, больше → select; диапазон с текущей серией подсвечен и выбран при открытии.
