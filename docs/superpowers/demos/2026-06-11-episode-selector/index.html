② История решений

  • 2026-06-05 — выбран триггер «Grid icon top-right» (AskUserQuestion, против вариантов в контрол-баре и в шапке). Панель = плавающий грид 5 колонок, геометрия source-панели.
  • 2026-06-11 (утро) — в селектор замаплены пользовательские данные: галочка «просмотрено» (anime-list + completed-rows), подчёркивание прогресса, тултип с названием серии; в шапку плеера — номер + название текущей серии.
  • 2026-06-11 (сегодня) — добавлена кнопка «Mark as watched» внутри селектора (паритет с Kodik) + resume-chip «Resume from MM:SS» (не автоперематывает — это решение пользователя).
  • Locked DS-правила — бейджи: inline = tinted, overlay = dark-glass + scrim; веса шрифта только font-medium/font-semibold; брендовые hue по провайдерам.
  • Сегодняшний вопрос — как селектор открывается и как выглядит внутри. Триггер 06-05 можно пересмотреть — данных в селекторе стало сильно больше (названия, прогресс, отметки).
