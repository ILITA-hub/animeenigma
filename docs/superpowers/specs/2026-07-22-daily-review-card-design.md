# Отзыв дня — редизайн карточки спотлайта

**Дата:** 2026-07-22 · **Статус:** утверждено владельцем (вариант «Линейка»)
**Файл:** `frontend/web/src/components/home/spotlight/cards/DailyReviewCard.vue`
**Прототип:** `.superpowers/brainstorm/review-card/content/review-card-v1.html`

## Что меняем

| Проблема | Решение |
|---|---|
| Единственная CTA «Читать все отзывы» ведёт в общий список отзывов | Кнопка удалена; адрес переехал ссылкой в подвал модалки |
| Нет пути «пойти смотреть» | **«Перейти к аниме»** — основная кнопка → `/anime/{id}` |
| Обрезанный текст — тупик, обрыв ничем не помечен | **«Читать полностью»** — вторичная кнопка, открывает модалку; рендерится **только когда текст реально обрезан** |
| Оценка — самый мелкий элемент кадра (12px пилюля справа в строке автора) | Цифра уходит влево от заголовка на дисплейный кегль с каноничным `◆` |
| Строка автора осиротела после выноса оценки | На место пилюли встала дата отзыва |

`/anime/:id/watch` — алиас, редиректящий на `/anime/:id`; «перейти» и «смотреть» — один адрес,
берём канонический (совпадает с FeaturedCard / CuratedCard).

## Вердикт-блок (вариант «Линейка»)

- Кегль `clamp(46px, 5vw, 64px)`, `font-display`, вес **600** (потолок DS), `tabular-nums`, трекинг `-.035em`, цифра белая.
- Ромб `ScoreDiamond` `size-3.5` в `text-cyan-400` — каноничный глиф оценки сайта (амбровая звезда зарезервирована за Shikimori/MAL).
- Знаменатель `/10` — `font-mono` 13px, `text-muted-foreground`, по базовой линии цифры.
- Разделитель — вертикальная волосяная линия 1px с растушёвкой по краям.
- `score === 0` → блок и линейка не рендерятся, заголовок занимает всю ширину.
- 390px: цифра 40px, зазор вердикта 14px.

## Кнопки

- **Перейти к аниме** — `router-link` + `buttonVariants({variant:'default',size:'md'})` → `/anime/{anime.id}`.
- **Читать полностью** — `<button>` + `buttonVariants({variant:'ghost',size:'md'})`, `v-if="isTruncated"`.
- Обе `text-sm`, в слоте `#cta` шелла, порядок фиксированный.

## Детект обрезки

- `isTruncated = el.scrollHeight > el.clientHeight + 1` на клампнутом `<blockquote>`.
- Замер на `onMounted` + `ResizeObserver` (ресайз окна, смена слайда, догрузка шрифта).
- `ResizeObserver` отключается в `onBeforeUnmount` — карточка живёт внутри `<Transition mode="out-in">`.
- Никакого подсчёта символов: кириллица, японский и переносы строк ломают любой порог.

## Модалка

- `<Modal>` из `@/components/ui`, `size="xl"`, дефолтный `modal=false` (не трогаем скролл-лок).
- Тело — `<ReviewMarkdown :source="data.review_text" />`: мини-разметка и спойлер-чипы. Никакого `v-html`.
- Шапка — заголовок аниме ссылкой; под ней вердикт + автор + дата.
- Подвал — ссылка «Все отзывы» → `/anime/{id}?ugc=reviews#section-comments`.
